# Sisyphus Design Agency 🎨

A premium, high-fidelity design agency landing page built with a focus on Swiss-Modern aesthetics and high-performance frontend engineering.


## ✨ Overview

Sisyphus is a professional design agency landing page that prioritizes clarity, geometric precision, and a minimalist user experience. Every element has been meticulously crafted to provide a polished, premium feel without the clutter of gratuitous animations or common design clichés.

## 🚀 Key Features

- **Swiss-Modern Aesthetic:** Clean lines, ample whitespace, and intentional typography.
- **High Performance:** Built on Next.js 15+ for lightning-fast delivery and optimal image processing.
- **Responsive Design:** Fluid transitions between desktop, tablet, and mobile views.
- **Custom Branding:** Fully vectorized geometric SVG logo integrated globally.
- **Polished Interactions:** Subtle, non-distracting visual feedback on interactive elements.

## 🛠️ Tech Stack

- **Framework:** Next.js (App Router)
- **Styling:** Tailwind CSS (Utility-first CSS)
- **Icons:** Lucide React
- **Typography:** Plus Jakarta Sans & Inter
- **Animations:** Framer Motion (entry interactions only)

## 📦 Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/your-username/sisyphus-agency.git
cd sisyphus-agency
```

### 2. Install dependencies
```bash
npm install
```

### 3. Run the development server
```bash
npm run dev
```

The application will be available at [http://localhost:3000](http://localhost:3000).

## 🎨 Design Philosophy

The project follows a **"Craftsmanship over Defaults"** philosophy. Instead of generic templates, it uses:
- **Precision Spacing:** Varied rhythm through custom padding and margins to prevent a "robotic" look.
- **Intentional Pairings:** Manrope and Inter font pairings to reinforce a technical but editorial mood.
- **Purposeful Logic:** Components like the dynamically inverting logo ensure brand consistency across different light/dark motifs.
